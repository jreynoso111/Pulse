# Project Instructions

These rules apply to every change in this repository.

## Core Rules
- Keep the project in English at all times.
- Do not introduce Spanish or mixed-language user-facing copy.
- Default all UI text, docs, comments, seeds, and placeholder content to English unless the user explicitly requests another language.
- Preserve these rules when editing existing files, adding features, or generating sample data.

## Product Context
- The project name is `Pulse`.
- Keep branding consistent as `Pulse` across the UI, docs, and seeded data unless the user asks for a different brand name.

## Data And Content
- Seed data should be realistic, internally consistent, and written in English.
- Support emails, workspace names, automation descriptions, and board descriptions should stay aligned with the `Pulse` brand.

## Implementation Notes
- If you find conflicting copy in the repository, translate or normalize it as part of the task.
- Treat this file as the source of truth for repository-wide writing conventions.
